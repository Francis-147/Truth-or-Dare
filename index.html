How to Play

1. click on the menu button to input your true and dare questions

2. Each input You make click on the ok button to add to true or dare list

3. click on the select button to select the number of players you want

4. click on the play button to start game

5. Each player should select a color

6. if the bottle points at any player after spining then the player will select either a true or a dare
